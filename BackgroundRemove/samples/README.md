# Samples — assets de prueba para perfeccionar el recorte

Dejá acá las imágenes reales con las que estás probando. Yo corro el CLI del
sidecar sobre cada una, miro los resultados y ajusto el algoritmo.

Ideal tener variedad para cubrir los casos:
- **logo sobre fondo plano** (ej. SAJARU BOX, KIFA) → método color
- **foto con persona/objeto** → método IA
- **ilustración** con fondo plano o degradado

Nombres sugeridos (cualquiera sirve, después los detecto):
- `sajaru.jpeg`
- `kifa.jpeg`
- `persona.jpg`

Cuando los dejes, avisame ("listo") y empiezo a testear + ajustar.
